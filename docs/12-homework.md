# Домашнее задание

## Задание 1. Новая колонка «Описание»

Добавьте в таблицу ещё одну колонку — короткое описание места.

1. Откройте `src/components/places/tables/placesTableColumns.tsx`.
2. Добавьте accessor-колонку по полю `description` (оно уже есть в типе
   `Place`). В шапке — «Описание».
3. В ячейке покажите только начало описания, чтобы таблица не «разъезжалась»:
   например, первые 60 символов и многоточие, если текст длиннее.
4. Сортировку для этой колонки не включайте (`enableSorting: false`) — сервер
   по описанию не сортирует.
5. Не забудьте `meta: { label: 'Описание' }`, чтобы колонка корректно
   подписалась в блоке управления видимостью.

## Задание 2. Кнопка «Сбросить выбранные строки»

Рядом со счётчиком выбранных строк добавьте кнопку, которая снимает выбор
со всех строк.

1. В `05-FinalPlacesTable.tsx` найдите блок `places-table-footer` со
   счётчиком «Выбрано строк: N».
2. Добавьте кнопку «Сбросить выбор». По клику вызовите готовый метод таблицы,
   который снимает выделение со всех строк (подсказка: у `table` есть метод
   для сброса выбора строк — поищите в подсказках IDE метод, начинающийся на
   `resetRowSelection`).
3. Покажите кнопку только тогда, когда есть что сбрасывать (когда выбранных
   строк больше нуля).
4. Добавьте немного CSS, чтобы кнопка не выбивалась из строки футера.

## Задание 3. Подсказка о скрытых колонках + запоминание (необязательное)

Сейчас, если пользователь скрыл колонки, об этом нигде не написано — можно
забыть, что часть данных спрятана. Сделайте это заметным.

1. В `05-FinalPlacesTable.tsx` посчитайте, сколько колонок сейчас скрыто.
   Подсказка: у таблицы есть методы, чтобы получить все колонки и проверить
   видимость каждой (`getAllLeafColumns`, `column.getIsVisible()`); скрытые —
   те, у которых `getIsVisible()` возвращает `false`, среди тех, что вообще
   можно скрывать (`getCanHide()`).
2. Рядом с блоком «Колонки» выведите подсказку на русском, например:
   «Скрыто колонок: 2» — и показывайте её только если скрытых колонок больше
   нуля.
3. Дополнительно покажите, **какие именно** колонки скрыты, перечислив их
   русские подписи (`column.columnDef.meta?.label`). Например:
   «Скрыто: Город, Год».
4. Проверьте, что после перезагрузки страницы подсказка соответствует
   сохранённой настройке (видимость колонок уже сохраняется в localStorage
   через `useColumnVisibilityStorage` — отдельно ничего сохранять не нужно,
   убедитесь, что это работает).
