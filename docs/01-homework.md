# Домашнее задание

Введение в React: компоненты, props, children, map, key

## Задание 1. Новый компонент

Создайте новый компонент `Greeting`, который принимает проп `name` (строка) и отображает приветствие, например:

```
Привет, Anna! Добро пожаловать в Travel Places.
```

Разместите компонент в папке `src/components/Greeting/` с отдельным CSS-файлом для стилей. Подключите `Greeting` в `App.tsx` так, чтобы он отображался между `Header` и списком карточек.

## Задание 2. Добавьте новые места

Добавьте в файл `src/data/places.ts` два новых места для путешествий (на ваш выбор). Убедитесь, что:

- каждое место имеет уникальный `id`
- заполнены все поля: `title`, `description`, `country`, `imageUrl`
- карточки корректно отображаются в списке без ошибок в консоли

Для поиска изображений вы можете использовать [Unsplash](https://unsplash.com/) — вставьте ссылку на подходящую фотографию в поле `imageUrl`.

## Задание 3. Компонент-обёртка с children

Создайте компонент `Section`, который:

- принимает проп `title` (строка) — заголовок секции
- принимает `children` — произвольное вложенное содержимое
- отображает заголовок `<h2>` и под ним всё, что передано через `children`

Используйте `Section` в `App.tsx`, чтобы обернуть список карточек. Например:

```tsx
<Section title="Популярные направления">
  <PlaceList places={places} />
</Section>
```

Добавьте для `Section` стили в отдельном CSS-файле: отступы, фоновый цвет или рамку — на ваше усмотрение.
